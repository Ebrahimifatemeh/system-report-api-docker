# System Report API with Docker Deployment

## Final Project Topics
 **System Report API**
- **Dockerized Web Server**

## Project Description
This project provides a simple API that returns system information such as CPU usage, memory usage, disk usage, hostname, operating system, and uptime.

The API is containerized using Docker, which connects the intermediate and advanced topics into one meaningful final project.

## Features
- REST API using Flask
- Returns live system information
- Dockerized deployment
- Simple and professional structure

## Technologies Used
- Python
- Flask
- psutil
- Docker

## Endpoints
- `/` → basic status message
- `/system` → system report in JSON format

## How the Topics Are Related
The **System Report API** is the main application, and the **Dockerized Web Server** is used to deploy and run this application in a containerized environment.
